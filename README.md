# Campus Connect

Campus Connect is a Next.js 15, Sanity, and Clerk-powered platform for campus communities. It provides features like community forums, marketplace, chat, and resource sharing for students and staff.

## Features
- Community forums and posts
- Marketplace for buying/selling
- Real-time chat
- Resource sharing
- User authentication with Clerk

## Getting Started

1. Clone the repository:
   ```sh
   git clone https://github.com/eshant-alfa/campusconnect-v2.git
   ```
2. Install dependencies:
   ```sh
   pnpm install
   ```
3. Set up environment variables (see `.env.example` if available).
4. Run the development server:
   ```sh
   pnpm dev
   ```
