# Campus Connect

Campus Connect is a purpose-built digital platform designed to foster vibrant, supportive, and engaged campus communities. Our mission is to empower students, staff, and organizations to connect, collaborate, and thrive through seamless communication, resource sharing, and community-driven features all in one place.

## 🌟 Mission

To create a safe, inclusive, and dynamic online environment where every member of the campus can:
- Build meaningful connections
- Access and share valuable resources
- Participate in campus life beyond the classroom

## 🎯 Key Features

- **Community Hubs:** Join or create campus communities for clubs, societies, academic groups, or interests. Each hub offers dedicated spaces for discussion, announcements, and events.
- **Marketplace:** Buy, sell, or exchange goods and services within your campus network, supporting sustainability and student entrepreneurship.
- **Real-Time Messaging:** Chat instantly with individuals or groups, making collaboration and socializing effortless.
- **Resource Center:** Discover and contribute to a curated collection of campus resources—maps, support services, study materials, and more.
- **Events & Activities:** Stay updated on campus happenings, RSVP to events, and never miss out on opportunities to get involved.
- **Verified Access:** Secure authentication with Clerk ensures only real campus members participate, keeping the community safe and relevant.
- **Moderation Tools:** Report and manage inappropriate content to maintain a respectful and positive environment.
- **Mobile-First Design:** Enjoy a responsive, accessible interface optimized for all devices.

## 🛠️ Technology Stack

- **Next.js 15** – Modern React framework for fast, scalable web apps
- **Sanity CMS** – Flexible, real-time content management
- **Clerk** – Secure authentication and user management
- **TypeScript** – End-to-end type safety
- **Tailwind CSS & Radix UI** – Customizable, accessible UI components

## 🚦 Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/eshant-alfa/campusconnect-v2.git
   cd campusconnect-v2
   ```
2. **Install dependencies:**
   ```sh
   pnpm install
   ```
3. **Set up environment variables:**
   - Copy `.env.example` to `.env.local` and fill in the required values for Sanity, Clerk, etc.
4. **Run the development server:**
   ```sh
   pnpm dev
   ```
5. **Access the app:**
   - Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🗂️ Project Structure

- `/app` – Application routes, pages, and API endpoints
- `/components` – Modular React components
- `/sanity` – Sanity schemas and utilities
- `/public` – Static assets
- `/action` – Server actions and business logic
- `/lib` – Utility functions and helpers
